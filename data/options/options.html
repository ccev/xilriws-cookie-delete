I. Tab Only
The addon will delete cookies when you close a tab. This is the default option. If you want to delete cookies when the browser is closed, please mark the - Session Only - option below.
II. Session Only
The addon will delete all cookies (for all tabs) only when you close the browser. Note: if you want to delete cookies for a tab when closed, please choose the above option instead.
